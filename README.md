# JavaScript Logic & Loops: The 50-Task Challenge

This repository contains a comprehensive set of **50 JavaScript programming exercises** focused on loops, nested structures, mathematical logic, and string manipulation. These tasks progress from basic sequence generation to complex number theory and pattern design.

---

## 🚀 Overview

The primary goal of this project is to master **control flow** and **algorithmic thinking** in JavaScript. Every task is implemented using core language features without relying on external libraries.

### Core Concepts Covered:
* **Looping Structures:** `for`, `while`, and `do-while` loops.
* **Nested Logic:** Multi-dimensional iteration for patterns and multiplication tables.
* **Digit Manipulation:** Extracting, summing, and reversing digits using string methods and mathematical operators.
* **Math Theory:** Prime number detection, Fibonacci sequences, Factorials, and Armstrong numbers.

---

## 📂 Project Sections

### 1. Fundamentals (Tasks 1–4, 13–15)
Covers basic iterations, including custom string joining, sum of ranges, and identifying even/odd sequences.

### 2. Geometric Patterns (Tasks 5–7, 12, 17–20)
Advanced use of **nested loops** to generate:
* Alpha-numeric grids.
* Right-angled and inverted triangles.
* **Floyd's Triangle** and diagonal matrices.

### 3. Practical Logic (Tasks 8–11, 16)
Classic programming challenges:
* **FizzBuzz** implementation.
* **Factorial** calculation via loops.
* **Fibonacci** sequence generation.
* Multiplication tables (1–5).

### 4. Advanced Digit Analysis (Tasks 21–50)
The "Heavy Lifters" — these tasks require deep logic to analyze numbers within specific ranges (e.g., 1–1000):
* **Reverse Logic:** Comparing numbers to their reversed versions.
* **Digit Property Checks:** Finding numbers where the digit sum equals the digit product.
* **Order Analysis:** Identifying numbers with ascending or descending digits.
* **Prime Reversals:** Numbers whose reverse form is a prime number.
